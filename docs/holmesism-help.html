How to Use Holmesism

1. Start an Investigation

  1. Tap the "+" button to create a new problem
  2. Enter a descriptive title and detailed description
  3. Select the problem category (Syntax, Logic, Runtime, etc.)
  4. Add any error messages or code snippets

2. Collect Evidence

  1. Go to the Evidence tab
  2. Add error messages, code snippets, log outputs, and observations
  3. Tag your evidence for easy organization
  4. Document test results and configuration details

3. Get AI Insights

  1. Visit the AI Insights tab
  2. Review AI-generated hypotheses with confidence ratings
  3. Consider evidence-based suggestions
  4. Use the insights to guide your investigation

4. Follow the Deductive Process

  1. Observe: Document everything systematically
  2. Hypothesize: Generate testable theories
  3. Test: Eliminate impossible causes
  4. Confirm: Document the solution and learn
